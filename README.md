# Divinity SkyBlock: Reborn
Продолжение карты Divinity SkyBlock с масштабным обновлением. В игре появилась генерация островов и биомной карты, новые боссы, предметы, артефакты, рецепты и многое другое!

![](https://github.com/smitexq/resources/blob/main/divinity_skyblock_reborn/img/prev.png)

Игра может содержать различные баги, ошибки и неккоректности в переводе. Пожалуйста, сообщайте о них в дискорд сервере, в соответствующем чате.

Discord: https://discord.gg/f6WfXqW6zU

Telegram: https://t.me/divinityskyblock

## Установка
- Скачайте репозиторий, распакуйте `.zip` архив
- Создайте мир, при создании выберите тип мира "Плоский", убрав все слои блоков, отключите генерацию строений
- Добавьте скачанный датапак `skyblock_pack` в `Наборы данных` и переместите его в правое окно
- Создайте мир, согласившись с предупреждением на экране
